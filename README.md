# JanusVLA
此为JanusVLA代码库。

所有人的代码需要上传至此仓库以保证正确计算代码与思路贡献量。

每一个人的代码头部需要包含以下内容。

```python
"""
file: <文件名>
author: <作者名>
date: <日期>
description: <描述>
version: <版本>
license: <协议>
Copyright (c) <年份> <作者名>
"""
```
若为c或者c语言风格注释的代码，则需要包含以下内容
```c
/***
file: <文件名>
author: <作者名>
date: <日期>
description: <描述>
version: <版本>
license: <协议>
Copyright (c) <年份> <作者名>
*/
```
提交代码时，请将代码提交至main分支，并提交pull request。
